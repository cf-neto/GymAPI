Bem-vindo ao GymApp


Sobre

O GymApp é uma aplicação web completa para gestão de academias, desenvolvida com foco em simplicidade, eficiência e escalabilidade. Ele oferece uma API RESTful para gerenciar membros, planos e check-ins, facilitando a integração com diversos frontends e sistemas.

O projeto foi construído utilizando tecnologias modernas, como ASP.NET Core com .NET para o backend, e Javavascript, CSS e TailwindCSS para o frontend. A ideia é oferecer uma solução robusta para academias de todos os portes.

Este projeto foi criado como parte de estudo sobre ASP.NET por Carlos Fernando, um desenvolvedor apaixonado por tecnologia e soluções Está aberto a contribuições e melhorias para expandir suas funcionalidades.

Endpoints

Membros (/members)

Método Rota Descrição
GET /members Lista todos os membros
GET /members/{id} Obtém um membro específico
POST /members/{id} Cria um novo membro
DELETE /members/{id} Remove um membro
PUT /members/{id} Atualiza um membro

Planos (/plans)

Método Rota Descrição
GET /plans Lista todos os planos
GET /plans/{id} Obtém um plano específico
POST /plans/{id} Cria um novo plano
DELETE /plans/{id} Remove um plano

Check-ins (/members/{memberId}/checkins)

Método Rota Descrição
GET /members/{memberId}/checkins Histórico de check-ins
POST /members/{memberId}/checkins Registra novo check-in
